¿Cuál es la diferencia entre Internet y la World Wide Web? El Internet es el medio por el que se accesa a la World Wide Web, que es donde está toda la información.
¿Cuáles son las partes de un URL? Protocolo, dominio, ruta y parámetros.
¿Cuál es el propósito de los métodos HTTP: GET, HEAD, POST, PUT, PATCH, DELETE? El propósito de los métodos es de acceso, publicación o actualización de archivos.
¿Qué método HTTP se debe utilizar al enviar un formulario HTML, por ejemplo cuando ingresas tu usuario y contraseña en algún sitio? ¿Por qué? Para enviar información sensible, lo mejor es usar un post ya que este cifra la información.
¿Qué método HTTP se utiliza cuando a través de un navegador web se accede a una página a través de un URL? Es un GET el método que se usaría.
Un servidor web devuelve una respuesta HTTP con código 200. ¿Qué significa esto? ¿Ocurrió algún error? Los códigos 200s significan que el método se ejecutó correctamente, por lo que no ocurrió ningún error.
¿Es responsabilidad del desarrollador corregir un sitio web si un usuario reporta que intentó acceder al sitio y se encontró con un error 404? ¿Por qué? Un error 404 es por parte del usuario, por lo que no se debe de arreglar la página web ya que no hay error.
¿Es responsabilidad del desarrollador corregir un sitio web si un usuario reporta que intentó acceder al sitio y se encontró con un error 500? ¿Por qué? Un error 500 sí es por parte del servidor, por lo que el equipo técnico debe corregir el error.
¿Qué significa que un atributo HTML5 esté depreciado o desaprobado (deprecated)? Menciona algunos elementos de HTML 4 que en HTML5 estén desaprobados. Significa que son tags que ya no se les da mantenimiento porque ya no están en uso. Algunos ejemplos de etiquetas depreciadas son < b >, < center >, < font > y < strike >.
¿Cuáles son las diferencias principales entre HTML 4 y HTML5? Las diferencias entre HTML 4 y 5 son que el 5 se usa ya solamente para dar estructura a la página, mientras que en el 4, se usaba para dar formato.
¿Qué componentes de estructura y estilo tiene una tabla? Los componentes de una tabla son, para definirla < table >, < tr > para definir las filas, < th > para los headers y < td > para definir el contenido de cada celda.
